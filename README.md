\# S3 Static Website Hosting



This project documents the creation of an S3 static website using AWS.



\## Objective

Host a static website in AWS S3 using best practices for security and accessibility.



\## Screenshots

See the `screenshots` folder for S3 bucket settings and website endpoint.



\## AWS CLI Commands

See `cli-commands/create-s3-commands.txt` for exact commands.



\## Terraform

Optional Terraform code to provision the S3 bucket and enable static website hosting.



\## Skills Demonstrated

\- AWS S3 management

\- Static website hosting

\- IAM and bucket policies

\- AWS CLI \& Terraform (infrastructure as code)
